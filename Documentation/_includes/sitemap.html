The rest after, in URL order, so output is deterministic. {%- endcomment %} {%- assign ordered = candidates | where_exp: "p", "p.nav_order" | sort: "nav_order" -%} {%- assign unordered = candidates | where_exp: "p", "p.nav_order == nil" | sort: "url" -%} {%- assign sorted = ordered | concat: unordered -%} {%- for p in sorted -%} {%- comment %} `assign` is page-global.
